Weekly Share CLI
=================

CLI tool for sharing URL to weekly.

Installation
------------

```
go get -u github.com/emir/weekly-share-cli
```

```
go install
```

Usage
------------

```
weekly u https://emirkarsiyakali.com/php-ve-visual-debt-goruntu-kirliligi-ecbb5267e412
```
